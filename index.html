How to Apply CSS to HTML

CSS file can be added by using <link> element winth in <head>.

How to Use JavaScript in a Webpage

External JavaScript file can be included by using <script> with src attribute in <head>.
Example:
<script src="script.js"></script>

Inline JavaScript can be written by using <script> followed by the JavaScript code.
Example:
<script>
console.log("This is inline JavaScript");
</script>
